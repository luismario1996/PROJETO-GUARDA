📚 Conceito (Nível Concurso)

Tipologia textual refere-se às diferentes formas de organização dos textos, classificadas de acordo com sua estrutura e finalidade comunicativa.

As principais tipologias textuais cobradas em concursos são:

  • Narrativa: relata fatos e acontecimentos (há ação no tempo).
  • Descritiva: caracteriza pessoas, objetos ou ambientes.
  • Dissertativa: expõe ideias, argumentos ou opiniões.
  • Expositiva: explica ou informa sobre determinado assunto.
  • Injuntiva: orienta, instrui ou dá ordens (manuais, receitas).

Em provas, é comum que o texto apresente mais de uma tipologia, mas sempre haverá uma predominante.

🧠 Explicação Simples (Metáfora + Estratégia de Prova)

Pense que cada tipo de texto é como um “modo de falar”:

  • 📖 Narrar → contar uma história
  • 🖼️ Descrever → tirar uma foto com palavras
  • 🧠 Dissertar → defender uma ideia
  • 📚 Expor → ensinar algo
  • 📢 Injuntar → mandar fazer algo

Agora vem o segredo de prova:

👉 Se tem ação, tempo e personagens → narrativa 👉 Se está “pintando” uma cena → descritiva 👉 Se tem opinião/argumento → dissertativa 👉 Se explica algo → expositiva 👉 Se manda fazer → injuntiva

⚠️ Dica de ouro: a banca adora misturar, mas sempre existe um tipo dominante.

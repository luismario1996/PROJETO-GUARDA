📚 Conceito (Nível Concurso)

As classes de palavras são categorias que agrupam os vocábulos da língua portuguesa de acordo com suas funções morfológicas e sintáticas.

As principais classes são:

  • Substantivo: nomeia seres, coisas ou ideias
  • Adjetivo: caracteriza o substantivo
  • Verbo: indica ação, estado ou fenômeno
  • Advérbio: modifica o verbo, adjetivo ou outro advérbio
  • Pronome: substitui ou acompanha o substantivo
  • Preposição: liga palavras
  • Conjunção: conecta orações
  • Interjeição: expressa emoção

🧠 Explicação Simples (Metáfora + Estratégia)

Pense em uma frase como uma equipe:

  • 👤 Substantivo → é o “personagem”
  • 🎨 Adjetivo → descreve o personagem
  • ⚡ Verbo → mostra o que ele faz
  • 📍 Advérbio → diz como, quando ou onde

Exemplo:

👉 “O policial rápido correu ontem.”

  • Policial → substantivo
  • Rápido → adjetivo
  • Correu → verbo
  • Ontem → advérbio

👉 Cada palavra tem uma função específica.

🎯 Estratégia de Prova

A banca NÃO quer que você decore tudo — ela quer que você reconheça a função.

Perguntas comuns:

  • “Qual a classe da palavra?”
  • “Qual função exerce na frase?”

👉 Dica:

  • Se indica ação → verbo
  • Se caracteriza → adjetivo
  • Se nomeia → substantivo
  • Se modifica → advérbio

⚠️ Pegadinhas de Concurso

A mesma palavra pode mudar de classe dependendo do contexto:

  • “O correr é saudável” → substantivo
  • “Ele vai correr” → verbo

👉 Isso cai MUITO.

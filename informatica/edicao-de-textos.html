📚 Conceito (Nível Concurso)

O Microsoft Office 365 é um conjunto de aplicativos utilizados para criação e edição de documentos, planilhas e apresentações.

Os principais programas são:

  • Word (textos)
  • Excel (planilhas)
  • PowerPoint (apresentações)

📌 Microsoft Word

Utilizado para criar e editar documentos de texto.

  • Formatação de texto (negrito, itálico)
  • Alinhamento
  • Inserção de imagens

📌 Microsoft Excel

Utilizado para cálculos e organização de dados em planilhas.

  • Uso de fórmulas
  • Funções (SOMA, MÉDIA)
  • Tabelas e gráficos

👉 Esse é o MAIS COBRADO em prova.

📌 Microsoft PowerPoint

Utilizado para criação de apresentações em slides.

  • Slides
  • Animações
  • Transições

🧠 Explicação Simples (Metáfora)

Pense no Office como uma equipe:

  • Word → escritor 📄
  • Excel → matemático 📊
  • PowerPoint → apresentador 🎤

🎯 Estratégia de Prova

  • Excel é o mais cobrado
  • Atalhos de teclado
  • Diferença entre programas

⚠️ Pegadinhas de Concurso

  • Word NÃO faz cálculos complexos
  • Excel NÃO é editor de texto
  • PowerPoint NÃO é planilha

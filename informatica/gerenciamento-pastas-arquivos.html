📚 Conceito (Nível Concurso)

Arquivos e pastas são utilizados para armazenar e organizar informações dentro do sistema operacional.

O gerenciamento é feito por meio do Explorador de Arquivos do Windows.

📌 Arquivo

Arquivo é um conjunto de dados armazenados no computador.

Exemplos:

  • .docx (Word)
  • .xlsx (Excel)
  • .pdf
  • .jpg

📌 Pasta

Pasta é um local usado para armazenar arquivos e outras pastas.

Serve para organizar o conteúdo.

📌 Operações com Arquivos

  • Criar
  • Copiar
  • Mover
  • Renomear
  • Excluir

📌 Caminho (Path)

O caminho indica a localização de um arquivo no sistema.

Exemplo:

👉 C:\Usuários\Documentos\arquivo.pdf

🧠 Explicação Simples (Metáfora)

Pense assim:

  • Pasta → gaveta
  • Arquivo → documento

👉 Você guarda documentos (arquivos) dentro da gaveta (pasta)

🎯 Estratégia de Prova

  • Diferença entre arquivo e pasta
  • Extensões de arquivos
  • Operações básicas

⚠️ Pegadinhas de Concurso

  • Pasta NÃO é arquivo
  • Excluir não apaga imediatamente (vai para lixeira)
  • Extensão define o tipo do arquivo

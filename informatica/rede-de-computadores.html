1.3 Redes De Computadores

Disciplinas Conteúdos Questões Simulados

📚 Conceito (Nível Concurso)

Redes de computadores são sistemas que permitem a comunicação entre dispositivos, possibilitando o compartilhamento de dados, recursos e serviços.

Exemplo: internet, redes locais (Wi-Fi), redes corporativas.

📌 Tipos de Redes

  • LAN: rede local (casa, empresa)
  • WAN: rede de longa distância (internet)
  • MAN: rede metropolitana

📌 Dispositivos de Rede

  • Roteador
  • Modem
  • Switch
  • Ponto de acesso (Wi-Fi)

📌 Endereço IP

O endereço IP é a identificação de um dispositivo na rede.

Exemplo:

👉 192.168.0.1

👉 Cada dispositivo tem um IP único.

🧠 Explicação Simples (Metáfora)

Pense na rede como uma cidade:

  • IP → endereço da casa
  • Roteador → central de distribuição
  • Internet → estrada que liga tudo

🎯 Estratégia de Prova

  • Diferença entre LAN e WAN
  • Função do roteador
  • Conceito de IP

⚠️ Pegadinhas de Concurso

  • Internet NÃO é LAN
  • IP não é fixo sempre (pode ser dinâmico)
  • Roteador ≠ modem

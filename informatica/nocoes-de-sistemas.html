📚 Conceito (Nível Concurso)

Sistema operacional é o software responsável por gerenciar o funcionamento do computador, controlando hardware e software.

O Windows é o sistema operacional mais utilizado no mundo.

📌 Funções do Sistema Operacional

  • Gerenciar arquivos e pastas
  • Controlar dispositivos (mouse, teclado, impressora)
  • Executar programas
  • Gerenciar memória

🧠 Explicação Simples (Metáfora)

Pense no sistema operacional como o “chefe do computador”.

👉 Ele organiza tudo 👉 Diz o que cada programa pode fazer 👉 Controla os recursos

📌 Elementos do Windows

  • Área de trabalho (Desktop)
  • Menu Iniciar
  • Barra de tarefas
  • Explorador de arquivos

🎯 Estratégia de Prova

  • Funções do Windows
  • Atalhos de teclado
  • Diferença entre arquivo e pasta

⚠️ Pegadinhas de Concurso

  • Sistema operacional NÃO é hardware
  • Windows é software, não computador
  • Pasta ≠ arquivo

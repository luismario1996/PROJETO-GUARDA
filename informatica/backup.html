📚 Conceito (Nível Concurso)

Backup é a cópia de segurança de dados, realizada para garantir a recuperação das informações em caso de perda, falha ou ataque.

📌 Importância do Backup

  • Evita perda de dados
  • Protege contra vírus e falhas
  • Permite recuperação de arquivos

📌 Tipos de Backup

  • Completo: copia todos os dados
  • Incremental: copia apenas alterações recentes
  • Diferencial: copia alterações desde o último completo

👉 Isso cai MUITO em prova.

📌 Onde Armazenar Backup

  • HD externo
  • Pendrive
  • Nuvem (cloud)

🧠 Explicação Simples (Metáfora)

Pense no backup como uma “cópia de chave”.

👉 Se perder a original 👉 Você ainda tem outra

🎯 Estratégia de Prova

  • Tipos de backup
  • Importância
  • Onde armazenar

⚠️ Pegadinhas de Concurso

  • Backup NÃO impede ataque
  • Backup serve para recuperação
  • Incremental ≠ completo

📚 Conceito (Nível Concurso)

Malware (software malicioso) é qualquer programa desenvolvido para causar danos, roubar dados ou comprometer sistemas.

📌 Principais Tipos de Malware

  • Vírus: precisa de ação do usuário para se espalhar
  • Worm: se espalha automaticamente pela rede
  • Trojan (Cavalo de Troia): se disfarça de programa legítimo
  • Spyware: espiona o usuário
  • Ransomware: bloqueia arquivos e cobra resgate

📌 Diferença entre Vírus e Worm

  • Vírus → precisa ser executado pelo usuário
  • Worm → se espalha sozinho

👉 Isso cai MUITO em prova.

📌 Formas de Infecção

  • Download de arquivos infectados
  • Anexos de e-mail
  • Links maliciosos
  • Dispositivos USB

🧠 Explicação Simples (Metáfora)

Pense assim:

  • Vírus → doença que precisa de contato
  • Worm → doença que se espalha sozinha
  • Trojan → presente falso 🎁

🎯 Estratégia de Prova

  • Diferença entre vírus e worm
  • Tipos de malware
  • Formas de infecção

⚠️ Pegadinhas de Concurso

  • Vírus NÃO se espalha sozinho
  • Worm NÃO precisa de usuário
  • Trojan parece programa legítimo

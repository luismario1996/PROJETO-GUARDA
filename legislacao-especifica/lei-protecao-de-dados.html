Lei Geral de Proteção de Dados

Lei nº 13.709/2018 (LGPD)

Disciplinas Conteúdos Questões Simulados

📚 Conceito

A Lei nº 13.709/2018, conhecida como LGPD, regula o tratamento de dados pessoais no Brasil, garantindo a proteção da privacidade dos cidadãos.

Ela estabelece regras sobre coleta, uso, armazenamento e compartilhamento de dados.

🧠 Explicação Simples

Pense na LGPD como a “lei que protege seus dados”.

👉 Empresas e órgãos usam seus dados 👉 Devem ter autorização e finalidade 👉 Você tem direitos sobre seus dados

📊 Tipos de Dados

  • Dados pessoais: nome, CPF, endereço
  • Dados sensíveis: saúde, religião, biometria

👉 Dados sensíveis têm proteção maior

⚖️ Princípios

  • Finalidade
  • Necessidade
  • Transparência
  • Segurança

👤 Direitos do Titular

  • Acesso aos dados
  • Correção
  • Exclusão
  • Informação sobre uso

🎯 Bizus de Concurso

  • Lei nº 13.709/2018
  • Protege dados pessoais
  • Dados sensíveis = proteção maior
  • Direitos do titular caem muito

⚠️ Pegadinhas de Prova

  • Nem todo dado é sensível
  • Uso sem finalidade é ilegal
  • Consentimento nem sempre é obrigatório (há exceções)
